# FisioCircuito v1.0.0

Applicazione web per la gestione di circuiti di fisioterapia con guida vocale e timer automatico.

## Come usare
1. Modifica il file `circuito.json` per personalizzare esercizi e tempi.
2. Apri l'indirizzo della GitHub Page sul tuo iPhone.
3. Lo schermo rimarrà acceso automaticamente durante l'uso.

## Funzionalità
- Lettura dinamica della configurazione JSON.
- Sintesi vocale in italiano.
- Supporto per esercizi a tempo (isometria) e a ripetizioni.
- Wake Lock API per prevenire lo spegnimento dello schermo.
